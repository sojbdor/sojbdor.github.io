How to use your webfont

  1. Upload the web font files to your server
  2. Add the code to your CSS stylesheet
    3.             
@font-face {
  font-family: 'simplement-regular';
  src: url('simplement-regular-webfont.woff2') format('woff2'),
  url('simplement-regular-webfont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

.fs-simplement-regular {
  font-family: 'simplement-regular';
}
  3. Apply the font family to your text(s)
    4.             
h1 {
  font-family: 'simplement-regular';
}
Remember to adjust the links to the web font files in the CSS if font files are located in a different folder.
